# Student Expense Tracker

## Problem Statement
Students often find it difficult to track their daily expenses and manage their money efficiently.

## Solution
This project is a simple web-based Student Expense Tracker that allows users to add daily expenses, view them in a table, and calculate the total expense. The data is stored using browser LocalStorage so that expenses are not lost on page refresh.

## Technologies Used
- HTML
- CSS
- JavaScript
- LocalStorage

## Features
- Add expense with date, category, and amount
- View all expenses in a table
- Automatically calculate total expense
- Persistent data storage using LocalStorage

## How to Run
1. Download or clone the repository
2. Open the project folder in VS Code
3. Open `index.html` using Live Server
   OR
4. Open `index.html` directly in any browser
